# 合同会社SP — සිංහල Deploy Guide

## 1) GitHub Pages (නිදහස්)
1. GitHub repo එකක් සාදන්න.
2. මේ ෆයිල් සියල්ල upload කරන්න.
3. **Settings → Pages** → **Deploy from a branch** → **main / root** තෝරන්න.
4. Published URL එකෙන් සයිට් එක බලන්න.

## 2) Netlify (ඉක්මන්)
1. netlify.com වෙත ලොග් වී "New site from Git" හෝ "Deploy manually" → ZIP upload කරන්න.
2. Contact form එක `thanks.html` වෙත redirect වේ.
